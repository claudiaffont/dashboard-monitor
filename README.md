Questo progetto è un simulatore di monitoraggio dati in tempo reale sviluppato per approfondire la Programmazione a Oggetti (OOP) e la gestione dinamica del terminale.

Funzionalità
Modellazione OOP: Utilizzo di classi per gestire sensori e dashboard.
Real-time Simulation: Generazione di dati casuali (temperatura, umidità) con aggiornamento ogni 2 secondi.
Data Management: Storico dei dati gestito tramite dizionari e liste per ogni sensore.
Interfaccia Terminale: Visualizzazione dinamica con barre di stato grafiche direttamente nella console.
 
Competenze Python applicate
Cicli while infiniti con gestione delle interruzioni (KeyboardInterrupt).
Utilizzo del costruttore __init__ e del parametro self.
Manipolazione di stringhe e formattazione f-string.
Gestione delle dipendenze standard (time, random).

Installazione
Clona il repository: git clone https://github.com/claudiaffont/dashboard-monitor.git
Avvia lo script: python live_data_monitor.py
